# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-02-06

### Added
- Initial release of Industrial Dashboard Suite
- Thermal Processing Dashboard with real-time monitoring
  - Temperature tracking (setpoint, actual, min, max)
  - Conveyor speed monitoring
  - 24-hour timeline visualization
  - Upheating period timer
- Production Tracking Dashboard
  - Hourly production logs
  - Target vs actual performance tracking
  - KPI cards with color-coded status
  - CSV export UI
  - Performance bar chart
- Core Framework Features
  - Extensible architecture for adding dashboards
  - Collapsible sidebar navigation
  - Full dark mode support
  - Responsive design (mobile, tablet, desktop)
  - Live data simulation
  - Professional wireframe aesthetic with Balsamiq Sans font
- Three placeholder dashboards (Energy, Quality, Inventory)
- Standalone HTML version (no build required)
- React artifact version for Claude.ai
- Full Vite + React project setup
- Comprehensive documentation
  - README.md with feature overview
  - SETUP.md with detailed instructions
  - CONTRIBUTING.md for contributors
  - LICENSE (MIT)

### Technical Stack
- React 18.2.0
- Tailwind CSS 3.3.6
- Lucide React 0.294.0
- Vite 5.0.8
- PostCSS & Autoprefixer

### Documentation
- README with badges and quick start
- Setup guide with 3 deployment options
- Contributing guidelines
- MIT License
- ESLint configuration
